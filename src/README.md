# About
This is a Rust TUI for viewing and selecting Markdown notes.

# Purpose
Anyone can use this tool to browse a directory of Markdown files, letting you fuzzy search for things
based on the directory name, content, and also search for metadata about those notes, stored as frontmatter
at the beginning of each note.

This tool is for managing these Markdown files with you being the one to actually edit the files
